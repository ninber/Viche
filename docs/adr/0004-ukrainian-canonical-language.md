# ADR 0004: Ukrainian Canonical Language

Status: proposed

## Decision

Use Ukrainian as the canonical language for domain text, authoritative resolutions, mandate materials, and official correspondence. Other languages are versioned translations.

## Consequences

- Localization must be built as a first-class layer.
- Translation state must be tracked separately from authoritative source text.
- Schema and workflow names should avoid locking business meaning into English-only UI labels.
